# Installation and Execution (Android)

This guide explains how to install and run the application on Android devices using an APK file.

## Languages
- 🇬🇧 English  
- 🇪🇸 Spanish → [README.es.md](README.es.md)  

## Download the APK
Download the `.apk` file from the repository.

## Install the application
- Open the `.apk` file on your Android device.
- If this is your first time installing apps outside of Google Play, the system will ask you to enable **unknown sources**:
  - Go to **Settings** → **Security**.
  - Enable installation from unknown sources (this may vary depending on the device).
- Confirm the installation when prompted.
- Wait for the process to complete.

## Run the application
You can open the application from:
- The app icon in the app drawer.
- The home screen shortcut (if added).

## System requirements
- Android 8.0 or later.
- Sufficient storage space.

## Tips
- Keep the APK file in case you need to reinstall the application.
