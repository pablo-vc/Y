#  Y – Escritorio

Este proyecto contiene el cliente de escritorio de **Y**, una aplicación multiplataforma de microblogging.

Este readme contiene toda la información relacionada con la parte de escritorio de la aplicación, incluyendo la cronología del desarrollo y las areas de investigación relevantes para el desarrollo copmleto del proyecto.

El cliente de escritorio permite a los usuarios interactuar con la aplicación a través de una aplicación en su ordenador.
